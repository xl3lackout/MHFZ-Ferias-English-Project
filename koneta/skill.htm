Point Breakthrough×Point Breakthrough10Increases the physical hitzone value of a part that has been hit repeatedly by 5.
The effect has a time limit and is only applied to a single part at a time.
If a part has been afflicted by this effect before, it will have a lower resistance to it and it will be easier to apply it to that part again.
※ The hit build up works like status, meaning it decays and it's harder to activate if you hit it less consistently.
※ All parts build up the effect separately, and hitting different parts will not reset it.
※ The skill duration cannot be extended by hitting the afflicted part.
※ Big multi hit attacks like pierce shots and LS piercing stab require a bigger number of hits.
※ Only the user benefits from the effect.
※ Fencing +2 and GL heatblade lowers the hit requirement. Also, reflect counts for part buildup but stylish up doesn't.
